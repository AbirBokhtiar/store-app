## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/AbirBokhtiar/store-app.git
   cd store-app
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Run the development server:**
   ```sh
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
